# touchSerialPortAt1200bps

## Description

This is a program to put into the bootloader for arduino and tinygo.
Touch the port at 1200bps.

## Requirement

## Usage

```
# windows
$ touchSerialPortAt1200bps.exe COM5

# ubuntu
$ touchSerialPortAt1200bps.exe /dev/ttyACM0
```
